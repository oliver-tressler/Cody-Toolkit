Image Properties

Image:

Name

Name of the entity image. Primarily used for distinguishing multiple images.

Image:

Entity Alias

The entity alias is used to identify the image among others within the EntityImageCollections of the Plugin Execution Context.

Image:

Plugin Stage Availability

Control wether this image is available in plugin executions happening before and/or after the database transaction. Not every step supports all availability types. At least one image type must be selected.

Image:

Pre-Image

The Image will contain the state of the entity as it was before the database transaction

Image:

Post-Image

The image will contain the state of the entity as it was after the transaction, when updates through user-input, workflows and plugins have been applied

Image Attributes

Image:

Attributes

Control which attributes will show up in plugins. If no attribute is selected in this list, the image will contain all attributes of this entity.
