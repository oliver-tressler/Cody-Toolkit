Assembly Properties

Assembly:

Name

Name of the assembly. Primarily used for distinguishing between multiple assemblies. The name is chosen automatically based on the DLL specified in the Location field.

Assembly:

Sandboxed

The assembly should run in a sandboxed context. This will limit the available functionality for plugins but enables logging via the ITracingService as well as simpler debugging.

Assembly:

Deployment

Choose where the assembly will be deployed. It is recommended to choose values other than Database only if you know what you're doing.
